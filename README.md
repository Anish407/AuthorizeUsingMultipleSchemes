## Authorization using different schemes

We have 3 authorization schemes
 - JWT1
 - Jwt2
 - Cookie 

<p>There are endpoints that can only be called using either JWT1, JWT2, Cookie , using any JWT token(JWT1 or JWT2) and more</p>
